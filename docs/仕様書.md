# Resemble Enhance ポータブル版 仕様書

## 1. はじめに
Resemble Enhance は、AI（機械学習）を活用した強力な音声補正・鮮明化ツールです。録音状態の悪い音声や、低音質の動画ファイルを高音質なオーディオへと蘇らせることができます。

## 2. 何ができるのか？（主な機能）

*   **高精度なノイズ除去 (Denoiser)**: 背景の雑音やホワイトノイズを AI が識別し、高品質に取り除きます。
*   **音声の鮮明化 (Enhancer)**: 低解像度の音声を高品位なサンプリングレート（44.1kHz）に拡張し、聞き取りやすく鮮明にします。
*   **動画・音声の一括処理**: フォルダ単位での処理が可能。サブフォルダの構造を維持したまま、動画（MP4, MKV, AVI, MOV）や音声（WAV, MP3, FLAC, M4A）をまとめて補正できます。
*   **プレビュー機能**: 最初の10秒間だけを先行して処理し、パラメータの効果をすぐに確認できます。
*   **形式変換・出力設定**: 出力形式（WAV / MP3）の選択、ファイル名末尾の指定（例: `_enhanced`）が可能です。

## 3. 使い方

1.  **入力フォルダの準備**: 補正したいファイル（動画・音声）を一つのフォルダにまとめます。
2.  **フォルダの指定**: 
    *   画面上の `入力フォルダ` 欄に、用意したフォルダをドラッグ＆ドロップします。
    *   `出力フォルダ` 欄に、処理後のファイルを保存したい場所をドラッグ＆ドロップします。
3.  **モードと設定の調整**:
    *   **自動モード**: AI が最適なバランスで補正を行います。
    *   **手動モード**: `ソルバー`、`NFE` (品質)、`Tau` (多様性)、`Denoise` (除去強度) の詳細設定が可能です。
    *   **出力設定**: 保存形式（WAV/MP3）やファイル名の末尾文字を指定します。
4.  **プレビュー (推奨)**: 「プレビュー (10秒)」ボタンを押し、短いサンプルで音質を確認します。
5.  **処理の開始**: 
    *   **一括補正開始**: フォルダ内の全ファイルを処理します。
    *   **停止**: 途中で処理を中断したい場合に使用します。
6.  **完了**: ログに「完了」と表示されたら、「開く」ボタンを押して出力フォルダを確認してください。

## 4. 特徴

*   **ポータブル・インストール不要**: レジストリを汚さず、このフォルダ一つで動作します。
*   **依存関係の自動セットアップ**: 初回起動時に必要な AI モデルや補助バイナリ（FFmpeg）が不足している場合、自動的にダウンロード・構築を行います。
*   **GPU 対応**: NVIDIA 製 GPU (CUDA) が搭載されている場合、高速な AI 処理が可能です。サイドバーから CPU / GPU を手動で切り替えることもできます。
*   **構造維持**: 入力フォルダ内のディレクトリ構造を維持したまま出力先に保存されます。
*   **リアルタイムログ**: 詳細な処理状況やエラーメッセージをコンソールで確認できます。

## 5. 動作に必要な環境とファイル

アプリケーションが正常に動作するために、以下の構成を維持してください。

| ファイル/ディレクトリ | 内容 |
|------|---------|
| `ResembleEnhance.exe` | ツール本体 |
| `bin/` | 音声処理用の補助バイナリ (FFmpeg) |
| `models/` | AI モデルデータ (Denoiser, Enhancer) |
| `仕様書.md` | 本ドキュメント |
